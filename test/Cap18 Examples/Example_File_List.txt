------------------------------------------------------------------------------------------------------------------------------------------------------
                                                       EXAMPLE FILES FOR CAP18 VERSION 6.2
------------------------------------------------------------------------------------------------------------------------------------------------------

The ZIP archive "CAP18_Example_Files.zip" is provided as part of the TxDOT Bent Cap Analysis (CAP18) software product. This archive contains the
example input data (.dat) files described in the CAP18 User Guide.  The archive also includes two Windows batch programs (.bat) files that may be
used to automate the processing of multiple CAP18 input files.

                                                                   Batch Files

       FileName                                                                Description

CAP18_Run_All.bat      This script is not an example batch program; it is a utility program that is used to run all input files that reside in a
                       specific directory.  DO NOT EDIT THIS FILE.  Detailed information about how to use this script is included in the file.
CAP18_Run_Examples.bat This script is used to process selected group of input files.  Firstly, copy and edit this file to use it.  Detailed
